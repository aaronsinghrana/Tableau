## ğŸŒ Global Health Insights Dashboard  

This dashboard explores global health trends using the Gapminder Health dataset. It combines multiple visualisations into a single interactive view to highlight disparities and long-term changes across continents and countries.

### Key Visuals and Insights  
- **Life Expectancy by Continent** shown using a bar chart to compare average outcomes  
- **Life Expectancy Trend over Time** displayed with a line chart to show long-term changes  
- **Population by Gender** visualised with a pie chart and interactive country filter  
- **Life Expectancy vs BMI** explored using a scatter plot coloured by continent  
- Additional visuals used to compare metrics such as blood pressure  

This dashboard highlights how life expectancy varies globally, with Europe showing the highest averages and Africa the lowest. Interactive filters allow users to focus on specific countries and trends over time.

### Key Visuals and Insights
